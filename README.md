# Item Warehouse for Diablo II LoD version 1.0.9.22

This repository contains item files (.d2i) for Diablo II LoD version 1.0.9.22
(so-called 1.09d)

## How to Use

Use any editor (e.g. [ATMA][1]) to import items (.d2i) into your saved
character files.
